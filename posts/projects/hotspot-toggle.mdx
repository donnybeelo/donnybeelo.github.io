Hotspot Toggle is a GNOME Shell extension that provides a convenient way to enable and disable your Android phone's mobile hotspot directly from your desktop's Quick Settings menu. By leveraging Bluetooth connectivity and NetworkManager integration, the extension automates the process of connecting to your phone's hotspot network.

## Overview

The extension works by establishing a Bluetooth connection with your Android device, which triggers the Automate flow on your phone to toggle the hotspot. Once the hotspot is active, the extension automatically connects your machine to the Wi-Fi network using NetworkManager. The entire process is managed through a simple toggle in GNOME Shell's Quick Settings panel.

This project started in August 2025 and has evolved from a JavaScript-based proof-of-concept to a fully-featured TypeScript extension with robust error handling, connection management, and localization support.

## Features

- **Quick Settings Integration**: Toggle your phone's hotspot directly from the GNOME Shell Quick Settings menu
- **Automatic Wi-Fi Connection**: Automatically connects to your phone's hotspot network when enabled
- **Visual Indicator**: Status indicator shows when connected to the hotspot network
- **Bluetooth-Based Control**: Uses Bluetooth connectivity to communicate with your Android device
- **Smart Connection Detection**: Intelligently checks if already connected to avoid unnecessary operations
- **Retry Logic**: Automatic retries for Wi-Fi connections with exponential backoff
- **Settings UI**: Easy-to-use preferences window for configuring Bluetooth MAC address and Wi-Fi SSID
- **Localization Support**: Currently supports English and Russian translations

## Technical Details

The extension is built using:

- **TypeScript**: Type-safe implementation for better maintainability
- **GNOME Shell Extension APIs**: Uses GJS (GNOME JavaScript) bindings
- **NetworkManager**: Integration via `nmcli` for Wi-Fi connection management
- **BlueZ D-Bus API**: Direct communication with Bluetooth stack for device management
- **Automate Flow**: Requires the [Automate flow](https://llamalab.com/automate/community/flows/50803) to be running on the Android device

## Installation

### Via GNOME Extensions Website

Install directly from [GNOME Extensions](https://extensions.gnome.org/extension/8508/hotspot-toggle/).

### Manual Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/donnybeelo/gnome-extensions-hotspot-toggle.git
   cd gnome-extensions-hotspot-toggle
   ```

2. Build and install:

   ```bash
   make install
   ```

3. Enable the extension using GNOME Extensions app or `gnome-extensions` CLI.

## Usage

### Initial Setup

1. **Pair Your Devices**: Ensure your Linux machine and Android phone are paired via Bluetooth.

2. **Configure Automate Flow**:

   - Install the [Automate flow](https://llamalab.com/automate/community/flows/50803) on your Android device
   - In the Automate app, set the Bluetooth device to your machine
   - Start the flow

3. **Configure the Extension**:

   - Open GNOME Extensions app
   - Find "Hotspot Toggle" and click the settings (cogwheel) icon
   - Enter your phone's Bluetooth MAC address (format: `00:11:22:33:44:55`)
   - Enter the SSID of your phone's hotspot network

4. **Connect to Hotspot**: Once configured, connect to the hotspot at least once manually to save the password in NetworkManager.
